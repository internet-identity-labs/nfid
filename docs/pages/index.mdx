## What is NFID Identity Kit?

NFID Identity Kit is a set of tools that allows you to easily integrate Identity provider into your application. It includes a set of ui components and helper functions to interact and request information about your application user.

 
## Usage

<Steps>
### Install

To use a **@nfid/identity-kit**, run the following command to install the dependencies:

```sh npm2yarn
npm i @nfid/identity-kit
```

### Import the library

```typescript
import { ConnectButton, ConnectModal } from '@nfid/identity-kit'


```
