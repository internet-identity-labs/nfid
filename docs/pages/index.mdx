## What is NFID?

NFID is a self-custodial identity provider and key management protocol that makes it easy for anyone
to make and use multi-chain accounts.

## What is NFID SDK?

The [`@nfid/embed`](https://www.npmjs.com/package/@nfid/embed) npm package is a JavaScript SDK that
easily integrates [NFID's Client Authentication Protocol](./how-it-works/auth-protocol) for user
authentication, registration, and transaction signatures.

Get started with [integration](./integration/quickstart).

## What is NFID Vaults?

NFID Vaults is the most powerful multi-approver (multi-sig) platform for ICP.
